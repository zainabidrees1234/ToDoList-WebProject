# ToDoList-WebProject

A simple and responsive Todo App built using HTML, CSS, JavaScript, and Bootstrap.

## Features

- Add new tasks
- Edit existing tasks
- Mark tasks as completed
- Filter tasks:
  - Today
  - Pending
  - Overdue
- Show / Hide completed tasks
- Clean and responsive user interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap 5
- Bootstrap Icons

## Project Structure

todo-app/
│── index.html
│── style.css
│── script.js

## How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open **index.html** in your browser

## Notes

- This project uses a JavaScript array (no database)
- Tasks are not stored permanently (they reset on refresh)
- Can be improved using localStorage
